Yeoman, Bower, and Gulp.js

A Lightweight Toolkit

What is Yeoman?

  • Yeoman is a toolkit that quickly generates the ecosystem for your project
  • Essentially it's a plugin that creates the scaffolding for your project
  • Creates robust and opinionated client-side stack
  • Using Yeoman and score together will create a faster and more efficient workflow

Benefits of Yeoman

  • Ability to easily create a well thought out build process
  • Great for rapid prototyping and working in teams
  • Comes with tools and frameworks that help devs quickly build beautiful web apps
  • Includes support for linting, testing, minifying and much more!
  • Developers can focus on solutions rather than worrying about little things

How does it work?

  • Yeoman is a npm package
  • Yeoman travels light and the initial installation does not come with any generators
  • Install generator that fits your project requirements or create your own generator for future use
  • Initialize generator in your project folder and let Yeoman do the heavy lifting for you!

Install Yeoman on your computer:

npm install -g yo

Install a generator to build your scaffolding:

npm install -g generator-foo

Initialize the generator in your project folder:

yo foo

Gulp.js

  • Build, preview, and test your project
  • Does everything precompiler gui's do plus more!
  • Thousands of options for cleaning up and organizing your code here
  • Example

Run this code to watch your project via your terminal:

npm install --save-dev gulp-watch
gulp-watch
gulp-watch example

Bower

  • Package manager
  • No longer have to manually download and manage packages
  • Vast majority of well-known plugins can be installed via Bower
  • Search through thousands of packages here

Run this code to download packages for your project:

bower install
